# 🧮 HorizonTechX Calculator App

A simple and responsive calculator built using HTML, CSS, and JavaScript.

---

## 🚀 Features

- Basic arithmetic operations ➕ ➖ ✖️ ➗
- Keyboard support ⌨️
- Clean and modern UI 🎨
- Responsive design 📱
- Clear (AC) and Delete (DL) functions

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

---

## 📂 Project Structure

📁 HorizonTechX_Calculator_App  
┣ 📄 index.html  
┣ 📄 style.css  
┗ 📄 script.js  

---

## ▶️ How to Run

1. Download or clone the repository  
2. Open `index.html` in your browser
